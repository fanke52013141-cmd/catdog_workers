# 热点选题与转译规则

本文件用于《呼噜噜看热点》栏目，也可用于其他栏目借热点起势。

## 1. 核心判断

热点不是伤害 IP，而是强化 IP。

原因：打工人本来就会在上班间隙刷热点、看热闹、转发梗图、用热点表达自己的情绪。呼噜噜的价值不是“报道热点”，而是把打工人看到热点时想说的话，用低电量打工喵的方式说出来。

一句话原则：

> 不是呼噜噜去蹭热点，而是热点经过呼噜噜，变成打工人的情绪表达。

## 2. 热点选择标准

优先选择符合以下条件的热点：

1. 打工人会关心：上班、下班、加班、工资、通勤、假期、年终、开会、职场关系、天气、物价、奶茶咖啡、外卖、睡眠、健康、情绪、周末生活、AI 工具和年轻人生活方式。
2. 能转译成打工人情绪：累、没电、想下班、想变有钱、想躺但不敢躺、想热爱生活但今天先活着。
3. 有视觉化空间：能变成发卡掉电、黑眼圈加重、咖啡回血、梦境穿越、变身、巨大特效。
4. 能说出打工人态度：不是复述热点，而是有呼噜噜的反应、啊呜呜的吐槽或打工人的共鸣结尾。
5. 能快速制作：轻热点适合 4-12 小时内发，大热点或魔幻特效可以 24-48 小时内发。

## 3. 热点分级

### A 级：必须优先做

与打工人强相关，且可快速共鸣。

- 假期调休、节前节后、周一、周五。
- 工资、年终奖、绩效、裁员焦虑、招聘。
- 天气极端：太热、太冷、暴雨通勤。
- 咖啡、奶茶、外卖、午休、熬夜。
- AI 工具、效率工具、职场软件。
- 全网流行但能被打工人化的梗。

### B 级：可做，但要转译

本身不是职场热点，但能被打工人情绪吸收。

- 影视剧名场面。
- 明星或综艺热梗。
- 流行语。
- 游戏/二次元/潮流梗。
- 生活方式话题。

转译方式：

- “如果打工喵也遇到这个……”
- “这个热点放到工位上就是……”
- “别人看到的是热闹，呼噜噜看到的是自己的周一。”

### C 级：谨慎做或不做

不容易转成呼噜噜视角，或者容易稀释账号。

- 纯八卦但没有打工人共鸣。
- 只适合信息搬运的热点。
- 过度沉重、灾难、事故、强社会冲突。
- 容易引战的议题。
- 低俗、擦边、恶意嘲讽类热点。

不是因为“伤 IP”，而是这些热点很难说出“打工喵替打工人表达”的价值。

## 4. 热点打分表

每个热点用 5 个维度打分，每项 1-5 分，总分 25 分。

| 维度 | 说明 |
| --- | --- |
| 热度 | 是否正在被大量讨论 |
| 打工人相关度 | 打工人是否真的关心 |
| 情绪共鸣 | 是否能表达累、没电、想下班、想生活变好 |
| IP 转译度 | 是否能自然变成呼噜噜/啊呜呜的反应 |
| 视觉表现力 | 是否适合 AI 变身、特效、梦境、毛绒动画 |

建议：

- 21-25 分：立刻做。
- 17-20 分：可做，优先做轻量版。
- 13-16 分：除非角度很好，否则先放。
- 12 分以下：不做。

## 5. 热点表达原则

热点表达形式不固定，不强行限制为某几种模板。

可用形式包括但不限于：

- 低电量吐槽。
- 魔幻变身。
- 梦境/穿越。
- 工位小剧场。
- 双人对话。
- 生活回血。
- 视觉特效短片。
- 伪纪录/观察日记。
- 表情包式短梗。

真正需要固化的不是形式，而是三件事：

1. 热点必须能被打工人关心或理解。
2. 热点必须经过呼噜噜的低电量视角转译。
3. 结尾最好回到 IP 记忆点、打工人态度或共鸣金句。

## 6. 标题模板

- “呼噜噜也刷到了这个热点”
- “这个热点，打工喵看完电量掉了 20%”
- “别人看热闹，呼噜噜看到了周一”
- “如果打工喵遇到最近很火的这个梗”
- “这个梗放到工位上，就不一样了”
- “打工喵低电量版：最近很火的那个”
- “热点看完了，咖啡也凉了”

## 7. 栏目边界

《呼噜噜看热点》不是新闻栏目，不负责完整解释热点。

它负责：

- 选取热点中的情绪。
- 用打工人视角表达。
- 用呼噜噜的低电量和啊呜呜的碎嘴完成转译。
- 让用户觉得：“对，我也是这么想的。”

## 8. 每日选题流程

1. 扫热点：抖音热榜、微博热搜、小红书趋势、B站热梗、节日节点。
2. 过滤：只留下打工人关心或可转译的热点。
3. 打分：按 5 个维度打分。
4. 定角度：确定“呼噜噜替打工人说什么”。
5. 定表达：根据热点选择最合适的形式，不预设模板。
6. 写脚本：控制在 1 分钟以内，普通内容建议 15-45 秒。
7. 发布后复盘：看评论里用户是否接上了打工人共鸣。
