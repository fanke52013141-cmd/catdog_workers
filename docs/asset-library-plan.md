# 可复用资产库规划

目标：把账号从“每条视频重新做”变成“调用固定 IP 资产 + 套用栏目模板 + 局部生成新镜头”。

## 1. 角色视觉资产

优先级：最高。

每个角色至少需要：

- 正面三视图：正面、侧面、背面。
- 半身标准图：用于封面、头像、评论区物料。
- 头像标准图：正常、开心、低电、宕机。
- 体型比例图：两人同框时的身高、头身比、体型差异。
- 服装设定：上班状态、家居状态、动物连体装状态。
- 道具设定：呼噜噜发卡/咖啡杯/工牌；啊呜呜尾巴/奶茶/零食袋。
- 色彩规范：主色、辅助色、禁用色。
- 负面约束：不能变成太精致、太成熟、太网红脸。

已确认视觉风格：3D 毛绒动画风。

呼噜噜已确认记忆点：

- 电量发卡。
- 猫猫毛绒外套。
- 黑眼圈。
- 左眼下方一颗痣。
- “躺平专员”工牌。
- 咖啡续命。

啊呜呜已确认记忆点：

- 狗骨头形状饼干。
- 嘴角总有零食碎屑。
- 小虎牙。
- 装满零食的包包。

## 2. 表情状态资产

呼噜噜表情九宫格：

- 正常低电。
- 发卡绿色满格。
- 发卡红色告急。
- 发卡灰色宕机。
- 喝到咖啡回血。
- 被催时委屈嘴硬。
- 假装在线。
- 突然虎气。
- 人生不易式躺平。

啊呜呜表情九宫格：

- 正常碎嘴。
- 看到好吃的眼睛发光。
- 尾巴狂摇。
- 饿到关机。
- 吃饱复活。
- 吐槽斜眼。
- 冲动上头。
- 嫌弃但关心。
- 吃饭专注到失联。

## 3. 动作姿态资产

呼噜噜高频动作：

- 从床上滚下来。
- 趴桌睡觉。
- 抱咖啡杯。
- 拖着身体走路。
- 发卡变红后缓慢趴下。
- 掏小本本立 flag。
- 突然站起来说“来，都给我上”。
- 三秒后坐回去。

啊呜呜高频动作：

- 尾巴狂摇。
- 边走边吃。
- 掏零食。
- 拍桌说“冲”。
- 拉着呼噜噜出门。
- 一边吃一边点头。
- 说话说到身体前倾。
- 饿到原地关机。

## 4. 场景资产

第一批核心场景：

- 公司工位：电脑、咖啡、奶茶、零食、工牌、文件。
- 会议室：适合开会、沉默、神评论。
- 公司楼下/打卡机：适合迟到、早餐、上班痛苦。
- 出租屋客厅：适合下班瘫倒、刷手机、立 flag。
- 出租屋卧室：适合早起失败、深夜内耗、宕机。
- 夜宵摊/便利店/奶茶店：适合回血、双人关系、商业合作。

每个场景建议包含：

- 空镜。
- 单人近景。
- 双人同框。
- 俯视/侧视参考。
- 可替换道具层。

## 5. 栏目模板资产

第一批栏目：

- 《呼噜噜看热点》：热点优先的流量入口，把热点翻译成打工人的低电量反应。
- 《呼噜噜的怪早晨》：魔幻变身/梦境/穿越栏目，每天醒来都变成一种打工人情绪隐喻。
- 《打工喵生存报告》：打工人生活合集，包含上班、下班、午休、加班、续命和小确幸。
- 《呼噜噜与啊呜呜》：双人对手戏合集，朋友分享、互相吐槽、一起回血，不要求每集出现啊呜呜。

推荐比例：

- 《呼噜噜看热点》：40%
- 《呼噜噜的怪早晨》：25%
- 《打工喵生存报告》：20%
- 《呼噜噜与啊呜呜》：15%
