# 封面海报风格建议

本文件用于统一“打工喵呼噜噜”抖音视频封面。目标是让主页看起来像一组连续的 IP 海报，而不是每条视频随手截帧。

## 1. 封面核心目标

封面要同时完成三件事：

1. 让用户 1 秒看懂这条和自己有关。
2. 强化呼噜噜的 IP 记忆点。
3. 保持主页视觉统一，形成“打工喵海报墙”。

## 2. 推荐方向

建议使用：

> 3D 毛绒角色大图 + 海报式大标题 + 栏目角标 + 电量状态符号。

不要做成复杂电影海报，也不要做成信息太多的营销图。封面应该“可爱、丧、清楚、有冲击”。

## 3. 基础版式

### A. 热点海报版

适合《呼噜噜看热点》。

- 上方：栏目角标，如“呼噜噜看热点”。
- 中间：呼噜噜大表情或魔幻画面。
- 下方/侧边：一句大标题。
- 角落：发卡电量状态或“躺平专员”工牌。

标题风格：

- “这个热点，电量掉了”
- “打工喵也刷到了”
- “别人看热闹，我看周一”

### B. 双人互动版

适合《呼噜噜与啊呜呜》。

- 左右双人构图：呼噜噜低电量，啊呜呜递零食/奶茶。
- 中间大标题制造关系冲突。
- 角落加小图标：咖啡 + 狗骨头饼干。

标题风格：

- “一个没电，一个饿了”
- “你睡你的，我说我的”
- “打工搭子续命局”

### C. 魔幻早晨版

适合《呼噜噜的怪早晨》。

- 画面更夸张：变成电池、困在表格、被闹钟追、黑眼圈倒计时。
- 标题直接说“今天变成了什么”。
- 可以用更强特效，但仍保留毛绒角色和发卡。

标题风格：

- “醒来只剩 3%”
- “今天被 Excel 困住”
- “黑眼圈开始倒计时”

### D. 生存报告版

适合《打工喵生存报告》。

- 画面偏日常：工位、通勤、午休、下班、便利店。
- 标题走共鸣，不追求太魔幻。
- 色彩更温和，留一点治愈感。

标题风格：

- “周一生存报告”
- “午休是精神急救”
- “下班后开始回血”

## 4. 固定视觉元素

每张封面尽量保留至少 2 个：

- 呼噜噜电量发卡。
- 黑眼圈。
- 左眼下方一颗痣。
- “躺平专员”工牌。
- 咖啡杯。
- 啊呜呜小虎牙。
- 狗骨头饼干。
- 啊呜呜零食包包。

## 5. 字体与标题规则

- 标题控制在 6-12 个字。
- 大字优先，不要堆长句。
- 一张封面只表达一个情绪。
- 标题要像评论区会说的话，不要像广告语。

推荐标题结构：

```text
对象 + 状态
```

例子：

- “周一又没电”
- “下班前来活”
- “班味实体化”
- “今天也在硬撑”

## 6. 色彩建议

建议形成栏目色，但不要过度花：

- 热点：红/橙小面积点缀，表示热度和警报。
- 怪早晨：蓝紫/荧光绿小面积点缀，表示魔幻和异常。
- 生存报告：米白/咖啡/浅灰，表示日常和疲惫。
- 双人互动：奶茶黄/暖橙，表示食物和朋友感。

角色主体仍以 3D 毛绒质感为主，不要让背景颜色压过角色。

## 7. 封面检查清单

- [ ] 1 秒能看懂主题。
- [ ] 标题不超过 12 个字。
- [ ] 至少出现 1 个呼噜噜核心记忆点。
- [ ] 栏目角标清楚。
- [ ] 主页连续看时风格统一。
- [ ] 不像纯营销海报，不像新闻截图。
