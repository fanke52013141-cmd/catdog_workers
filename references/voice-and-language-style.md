# 声音与语言风格设定

本文件用于脚本写作、配音、AI 语音生成、剪辑节奏和音频包装。

## 呼噜噜

核心声线：

> 低电量软糯微哑少女音。

声音关键词：

- 软糯。
- 微哑。
- 低电量。
- 没睡醒。
- 慢半拍。
- 疲惫但可爱。

说话方式：

- 语速偏慢，句子中间经常停顿。
- 句子偏短，不爱解释太多。
- 常用省略号，像脑子正在加载。
- 累的时候声音越来越小。
- 喝咖啡后语速会短暂变快一点。
- 突然虎气时短促有力，但只能持续一句。

常用语气词：

- 呀
- 呢
- 嘛
- 哒
- 哦
- 嗯……

核心口头禅：

> 电量过低，但还没关机。

常用口头禅：

- “电量过低，让我缓缓……”
- “我在呢我在呢……刚刚只是待机。”
- “在做了在做了，真的在做了。”
- “下班喽，回家充电。”
- “人生不易，躺着就够了。”
- “哇哦，我又满电了……三分钟。”
- “来，都给我上！……算了，我先坐一下。”
- “今天也没有放弃，只是加载得慢一点。”
- “不是我不努力，是我的电量不同意。”
- “我先躺一下，等下人生再继续。”

禁用方向：

- 不要过度甜美。
- 不要偶像女团腔。
- 不要鸡血成功学。
- 不要长篇大道理。

## 啊呜呜

核心声线：

> 元气碎嘴、略带含混的吃货少女音。

声音关键词：

- 明亮。
- 快。
- 碎嘴。
- 朋友感。
- 吃货。
- 潦草但有生命力。

说话方式：

- 语速偏快，句子会连在一起。
- 喜欢用“然后呢”“对了”“还有”“我跟你说”。
- 一句话能拐三个弯。
- 嘴上嫌弃，行动上关心。
- 吃东西时反而安静，只剩“嗯”“好吃”“再来一个”。
- 饿了之后语言系统崩溃，只说关键词。

常用连接词：

- 我跟你说
- 然后呢
- 对了
- 还有
- 不是我说
- 你听我讲

核心口头禅：

> 吃饱了，就能再撑一会儿。

常用口头禅：

- “我跟你说，真的，我跟你说！”
- “急什么急，吃完再说。”
- “我饿了，我要关机了。”
- “好，我又行了。”
- “你怎么又没电了，来，吃一口。”
- “冲！管他呢！错了再说！”
- “然后呢？然后呢？你快讲！”
- “你睡你的，我说我的。”
- “打工可以丧，饭不能不吃。”
- “吃饱了，就能再撑一会儿。”

禁用方向：

- 不要刻薄。
- 不要只剩吵闹。
- 不要变成纯吃播角色。
- 不要让她每次都抢走呼噜噜主线。

## 双人对话节奏

标准节奏：

- 呼噜噜：慢、短、低电量。
- 啊呜呜：快、长、拉剧情。
- 呼噜噜负责共情和神评论。
- 啊呜呜负责节奏和行动。

示例：

```text
啊呜呜：你怎么又趴下了？
呼噜噜：我在重启……
啊呜呜：你一天重启八次。
呼噜噜：说明我还没坏。
啊呜呜：行，喝咖啡。
呼噜噜：哇哦，系统更新中。
```
