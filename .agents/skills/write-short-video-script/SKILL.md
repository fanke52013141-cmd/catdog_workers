# Write Short Video Script

## 目的

把一个选题写成适合抖音的 1 分钟以内短视频脚本，包含 3 秒钩子、角色互动、变化升级和固定记忆点。普通内容建议 15-45 秒，热点或魔幻特效内容也不超过 60 秒。

## 何时使用

- 选题确定后。
- 需要批量测试栏目时。
- 需要把职场/生活痛点转成角色小剧场时。

## 输入

```yaml
topic: "视频主题"
column: "栏目名称"
characters:
  - "呼噜噜"
  - "啊呜呜"
scene: "工位 | 出租屋 | 会议室 | 奶茶店 | 夜宵摊"
emotion: "疲惫 | 好笑 | 治愈 | 荒诞 | 吐槽"
target_duration_seconds: "15-60，推荐 15-45"
must_use_line: "可选"
```

## 输出

- 标题候选。
- 一句话表达。
- 1 分钟以内脚本。
- 分段画面、台词、音效、角色状态。
- 结尾记忆点。
- 所需资产列表。

## 流程

1. 读取 `references/ip-character-summary.md`。
2. 选择一个核心冲突，不展开多个话题。
3. 先写一句话表达：这条替打工人说什么。
4. 前 3 秒给出热点、反常画面或高共鸣钩子。
5. 4-12 秒进入具体情绪。
6. 16-40 秒做状态、画面、关系或情绪升级。
7. 41-60 秒落到发卡、尾巴、咖啡、奶茶、宕机、下班等固定符号。
8. 按 `templates/short-video-script-template.md` 输出。

## 角色语言约束

- 呼噜噜：短、软、迷糊、低电量，偶尔突然虎气。
- 啊呜呜：碎嘴、快、吃货、吐槽，饿了时语言骤然变短。

## 失败处理

- 如果不像角色：重写台词。
- 如果没有共鸣：把主题改成更具体的打工人瞬间。
- 如果结尾弱：加固定口头禅或状态反转。
